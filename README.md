# Аванпроект: двоступеневий підвіс для мультиспектрального сенсора БПЛА (Мехатронні системи)

**Курс:** Мехатронні системи  
**Викладач:** к.т.н., старший викладач, Іван Володимирович Жежера  
**Тема лабораторної роботи:** Аналіз становища та створення аванпроекту  

## 1. Обраний варіант ТЗ
**Варіант:** №4 — підвіс для сільськогосподарського коптера (аналіз стану рослин).  
**Завдання:** обрати мультиспектральну камеру для стійкого знімання з 60–100 м, підібрати двигуни двоступеневого підвісу та плату керування з підтримкою **S.BUS** та **UART**, сформувати очікувані технічні характеристики, виконати ескіз/креслення в CAD та схеми в Draw.io.

## 2. Короткий опис моделі
Проєктується **двоступеневий (Pitch/Roll) підвіс прямого приводу** для стабілізації мультиспектральної камери під час польоту. Мета — компенсувати вібрації та кутові коливання коптера, утримувати оптичну вісь у напрямку “в надир” для отримання даних, придатних для агроаналітики (водний баланс, азот/хлорофіл, біомаса). Керування підвісом виконується платою стабілізації, що приймає команди з польотного контролера по **S.BUS** або через **UART**.

## 3. Аналіз потреб та актуальність
У задачах агромоніторингу критичними є повторюваність зйомки, стабільність кута нахилу сенсора, мінімізація змазу та геометричних викривлень. Інтегровані комерційні рішення зручні для старту, але обмежують вибір сенсорів/інтерфейсів і не завжди дозволяють встановлення спеціалізованого обладнання. Кастомний підвіс дає можливість адаптувати конструкцію під конкретну камеру (маса, габарити, центр мас), забезпечити сумісність із обраним польотним контролером та спростити технічне обслуговування (демпфери, кріплення, кабель-менеджмент).

## 6. Очікувані технічні характеристики (цільові)
- Робоча висота зйомки: **60–100 м**.  
- Ступені стабілізації: **2 осі (Pitch/Roll)**.  
- Режими керування: RC-режими/кути через **S.BUS**, сервіс/налаштування/телеметрія через **UART** (за потреби).  
- Корисне навантаження: **камера до 350 г** (включно з кріпленням/кабелем; уточнюється після CAD-розкладки).  
- Орієнтовна маса підвісу (без камери): **TBD** (після побудови моделі та підбору матеріалів).  
- Діапазон кутів (проектно): Pitch **±45°**, Roll **±45°** (уточнюється під механіку та обмеження кабелів).  
- Конструкція: демпферна платформа + жорсткі рами осей, фіксація кабелів для зменшення паразитних моментів, можливість швидкого доступу до сенсора для сервісу.

## 7. Доцільність розробки моделі
Розробка є доцільною, оскільки забезпечує:
1) сумісність із конкретною мультиспектральною камерою та нестандартними сенсорами;  
2) оптимізацію маси/габаритів під конкретний коптер і політний профіль;  
3) керування від польотного контролера через **S.BUS/UART** без прив’язки до закритої екосистеми;  
4) ремонтопридатність і модульність (демпфери, кріплення, площадка сенсора);  
5) підвищення якості даних (стабілізація положення сенсора для надійної аналітики).
