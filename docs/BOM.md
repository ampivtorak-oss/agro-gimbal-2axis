## 7) Очікувані технічні характеристики (для звіту ЛР-1)
- Ступені свободи: **2 осі (Pitch/Roll)**  
- Тип приводу: **Direct-drive BLDC (gimbal motors)**  
- Режими: **стабілізація по Pitch/Roll; керування кутами від FC**  
- Робочий діапазон кутів (ціль): **Pitch ±45°, Roll ±45°** (фіналізувати після CAD, щоб уникнути зачепів)  
- Канали керування: **UART + RC-канал**, Trigger/Sync — окремий канал  
- Живлення: **від бортової шини БПЛА через UBEC/DC-DC**, з фільтрацією від EMI
